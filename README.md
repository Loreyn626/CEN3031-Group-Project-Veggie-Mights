## Project Setup

1. Clone the repository

2. Install dependencies:

 - Install node.js version ^20.19.0 || >=22.12.0
 
 Run the following: 

 ```
 npm install
 pip install -r requirements.txt
 cd frontend
 npm install
 ```
3. In separate terminals, run:
```
# in ./frontend
npm run dev

# in ./
npm run dev

# in
python mapAPI.py
```
4. Access project through http://localhost:5173/
